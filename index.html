Important things to include for SEO

  • Title tag: Should be short and concise.
  • Use meta tag viewport to size correctly on all divices
  • Meta tags for keywords, author, and description
  • Maybe need og tags for social media.
  • robots meta tag instructs search engines not to show the page in search results.
  • Use alt attribute help index images for search engines.

6 Different font sizes

  • Font size 1. Font value: 30px. When you know the exact font that is required or used to.
  • Font size 2. Font value: smaller. When you want to make font salller then currently existing font.
  • Font size 3. Font value: larger. When you want to make font larger then currently existing font.
  • Font size 4. Font value: 1em. Em compounds so can have different font in one sentence. If this element has no ancestors, em font size will equal to broswer default font size.
  • Font size 5. Font value: 1.2rem. Act same as em but with no compounds problem.
  • Font size 6. Font value: 130%. Used when want text changed by certain percentage.
